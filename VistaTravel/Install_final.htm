Finishing the installation:

To finish the installation of VistaVacations please follow the steps below.

- Restart IIS.
- Open the IIS Admin tool and browse to the AirlineService Web using the tree view.
- Right click on the web and open the "properties".
- Under Application Settings:
- Click Create
- Set the Application Protection to Low
- Click Configuration -In the App Config dialog, Choose Add
- Select the AirlineServiceISAPI.dll (you have to change the file type from the default .exe)
- Enter ".dll" for the Extension
- Click OK, Apply, OK, etc. to get out of the dialogs

Now you can browse the VistaVacations web on this machine at http://localhost/VistaVacations/Index.aspx.

Opening Solution Files:

-If you want to open the solution files you must first configure the server extensions on the LodgingMgr, POILib, and VistaVacations webs.  If you are unsure of how to do this see Configuring Server Extensions.

-Finally, you may double click on the solution files (denoted by the .sln extension) in your wwwroot directory that you specified to view the code.
