There are two more steps to complete the installation of the VistaVacations sample application:

  1. Install the SQL Server Databases
  2. Install the VistaVactions Web as well as the related webservices

Please note before proceding: You will need to know the System Administrator user name and password for the SQL Server to install the database (step 1). Also, you will need to have administrator privileges on the Windows 2000 machine in order to install the web pages (step 2).
