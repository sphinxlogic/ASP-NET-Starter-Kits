Configuring Server Extensions
   


1.  Open the IIS Admin tool by going to your start menu - programs - Administrative Tools - Internet Services Manager

2.  Expand the tree view control with your server name.  Then expand the node labeled "Default Web Site".

3.  Under this node there will be the webs labeled "LodgingMgr", "POILib", and "VistaVacations".  For each of these webs do the following:

-Right Click, choose "All Tasks" and click "Configure Server Extensions".  This will launch a wizzard.

-Click the "Next" button on the first page.

-Click the "Next" button on the second page.

-Click the "Next" button on the third page.

-Click the "Finish" button on the last page.

4.  That is all there is too it!  Your webs have now been configured with server extensions.
